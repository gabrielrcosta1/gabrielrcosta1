# Gabriel Ribeiro

Backend Software Engineer from Brazil 🇧🇷

I build scalable APIs, distributed backend services, and enterprise applications with a strong focus on clean architecture, performance, maintainability, and developer experience.

My primary expertise is **PHP** and the **Laravel** ecosystem, where I've spent the last several years designing REST APIs, authentication systems, background jobs, queues, integrations, and scalable business applications.

I also build backend services using **Python (FastAPI)**, **C# (.NET)**, **Go (Golang)**, and **Node.js (NestJS)**, choosing the right technology based on each project's requirements.

Currently exploring **Elixir** and **Phoenix** to expand my knowledge of highly concurrent and fault-tolerant distributed systems.
